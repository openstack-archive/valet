=========
Valet API
=========

Valet gives OpenStack the ability to optimize cloud resources while
simultaneously meeting a cloud application's QoS requirements. Valet is
composed of an api service, an optimizer (Ostro), and a set of OpenStack
plugins.

See the API documentation for a complete list of supported requests.
